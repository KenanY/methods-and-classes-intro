Introduction to Methods

A method is a collection of statements. A method may be predefined, such as by the Java system. However, you'll need to eventually create your own methods.

User-defined methods in Java fall into two categories:

  • Value-returning methods—methods that have a return data type. The return statement will be explained later.
  • Void methods—methods that do not have a return data type.

The name, parameters, and data type of a method are part of the heading.

For example, a heading might look like this:

public static int abs(int number)

The code of a method is called the body of the method. Together, the heading and the body form the definition of the method.


public static int abs(int number)
{
    if (number < 0)
        number = -number;

    return number;
}

The variable called in the heading is called the formal parameter of the method.

Syntax: Method

The syntax of a method is:


modifiers(s) returnType methodName(formal parameter list)
{
    statement(s)
}

return Statement

A value-returning method uses a return statement to return its value.


return expr;

When the statment executes in a method, the method immediately terminates.

Value-returning Method


public static double square(double num)
{
    return num * num;
}

num1 = 8.0;
num2 = square(num1);

Void Method

Used for methods that do not return a value.


public static void printNum(double x)
{
    System.out.println(x);
}

Classes


public final class Math
{
    public static final int dos = 2;
    public static final int tres = 3;

    public static double abs (double a)
    {
        // code to find the absolute value
    }
    public static int max (int a, int b)
    {
        // code to find the maximum of a and b
    }
    public static double pow (double a, double b)
    {
        // code to find the a to the power b
    }
}

The members of a class fall into four catergories:

  • default—This type of visibility should be avoided most of the time. It allows the member to be accessed from anywhere within the package.
  • private—You cannot access it directly outside the class.
  • public—You can access it directly outside the class.
  • protected—Explained in a later chapter.

Constructors

A constructor is a special type of method within a class. It has the same name as the class. Constructors execute automatically when an object of that class is created. If there are multiple contructors, each much have different formal parameters, or else they'd just be duplicates of each other.

Variable Declaration and Object Instantiation

You can declare reference variables of classes:


Thermometer myTemp;
Thermometer yourTemp;

To store data, you need to use the new operator:


myTemp = new Thermometer();
yourTemp = new Thermometer(35);
