Introduction to Methods

A method is a collection of statements. A method may be predefined, such as by the Java system. However, you'll need to eventually create your own methods.

User-defined methods in Java fall into two categories:

  • Value-returning methods—methods that have a return data type. The return statement will be explained later.
  • Void methods—methods that do not have a return data type.

The name, parameters, and data type of a method are part of the heading.


public static int abs(int number)
{
    if (number < 0)
        number = -number;

    return number;
}

The variable called in the heading is called the formal parameter of the method
